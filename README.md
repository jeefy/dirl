# DIRL (Dungeons, IRL)
Simple dungeon map manager for IRL games. Ideal for TV-tables, projectors, and Chromecasting.

## Dependencies
* nodejs v9.11+
* npm 6.1.0+
* ng 6.0.8+

## Quickstart
```
git clone https://github.com/jeefy/dirl.git
cd dirl/angular
ng serve
```
Open a browser to `localhost:4200`
